# Matrix Operations Tool  
A simple and interactive Python application that performs basic matrix operations using the **NumPy** library.  
This project allows users to input matrices of any size and execute operations like addition, subtraction, multiplication, transpose, and determinant calculation.

---

##  Features
- **Matrix Input:** Accepts matrices of user-defined rows and columns.
- **Matrix Addition**
- **Matrix Subtraction**
- **Matrix Multiplication**
- **Transpose of Matrix**
- **Determinant Calculation**
- **Interactive Console-Based Menu**

---

##  Technologies Used
- **Python 3**
- **NumPy Library**

## Requirments

1.Install NumPy
pip install numpy

2. Running the Program
python matrix.py
